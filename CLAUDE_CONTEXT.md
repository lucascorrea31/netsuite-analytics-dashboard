# NetSuite Analytics Dashboard - Development Context

## Project Overview
Building a NetSuite Analytics Warehouse (NSAW) dashboard with Next.js 14.
Focus: Transform slow NSAW reports into fast, cached dashboards.

## Current Phase
Initial setup - Turborepo monorepo structure created.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Turborepo + pnpm
- OAuth 2.0 for NetSuite
- Redis for caching
- Recharts for visualization

## Reports to Build
1. Sales by period
2. Current inventory
3. Customer performance
